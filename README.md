Quick start
===========
`git clone git@github.com:luongthanhlam/react-boilerplate.git`

`cd react-boilerplate`

`cp src/config.json.example  src/config.json`

`npm i`

### Run Webpack

`npm run start`

### Run Graphiql
`npm run graphiql`
